The Raspberry Pi Computer

What is the Raspberry Pi?

The Raspberry Pi 2 is a powerful single board computer from the Raspberry Pi foundation.

It comes with

  • BCM2836 System On A Chip (SoC) consisting of ARM7 quad core CPU running at 900Mhz per core.
  • 1GB or DDR2 RAM running at 450Mhz
  • 4x USB 2 Ports
  • 1x 10/100 Fast Ethernet
